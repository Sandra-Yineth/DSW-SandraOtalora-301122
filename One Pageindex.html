¿QUE ES CSS?

CSS es un lenguaje de estilos para dotar de aspecto visual y estético a una página web. Una página web es realimente un documento de texto; en dicho documento se escribe código HTML, en el que se crea el contenido de una web. Por otro lado existe el código CSS, que unido al código HTMLpermite darle forma, color, posición, insertar imágenes, entre otras características visuales a un documento web. En resumen, se trata de un idioma o lenguaje, como podría ser el inglés o Francés, que otros navegadores web como Chrome o Firefox conocen y pueden entender.

¿QUÉ ES CSS Grid?

Es un sistema de maquetación que se utiliza en páginas web y que permite crear un loyaut o estructuras para páginas web a actuales. Grid CSS nace de la necesidad de obtener mejoras y nuevas características que permitieran crear cuadriculas flexibles y potetentes de forma instantánea con unan nueva familia de propiedades CSS.

Para crear diseños basados en GRID CSS es necesario tener en cuenta los siguientes conceptos:

    Contenedor:El elemento padre contenedor que definirá la cuadricula o rejilla.
    Item:Cada uno de los hijos que contiene la cuadricula (elemento contenedor)
    Celda:(Grid Celd). Cada uno de los cuadritos (unidad mínima de cuadricula) de la cuadricula.
    Área:(Gris area) region o conjunto de celdas de la cuadricula.
    Banda:(Grid track) banda horizontal o vertical de celdas de la cuadricula.
    Línea:(Grid line) Separador horizontal o vertical de las celdas o cuadricula.
